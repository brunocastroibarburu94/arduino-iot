# PlatformIO projects

### Useful shell commands

```bash
# On a Command Prompt shell

# Build project
C:\Users\bruno\.platformio\penv\Scripts\platformio.exe run

# Upload project to board
C:\Users\bruno\.platformio\penv\Scripts\platformio.exe run --target upload --environment megaADK 

# Monitor Serial communication
C:\Users\bruno\.platformio\penv\Scripts\platformio.exe device monitor --environment megaADK 
```
